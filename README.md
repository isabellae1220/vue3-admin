# vue3-admin

一个 Vue 3 + Vite + TypeScript 的后台管理系统学习项目。

## 技术栈

- Vue 3 (Composition API + `<script setup>`)
- Vite 8
- TypeScript
- Vue Router 4
- Pinia
- Element Plus
- Axios
- ECharts
- Apifox (Mock 数据)

## 核心功能

- [ ] 登录 + 权限管理
- [ ] 动态路由(基于角色)
- [ ] 路由守卫 + 404 页面
- [ ] 用户管理(增删改查 + 分页 + 搜索)
- [ ] 面包屑导航
- [ ] Tag 标签栏
- [ ] ECharts 数据可视化
- [ ] Axios 封装 + Mock 数据
- [ ] 多环境配置
